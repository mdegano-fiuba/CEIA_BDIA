## Set Up

Ejecutar el siguiente comando para generar el entorno (PostgreSQL + pgAdmin):

```Bash
docker-compose up -d
```

Ejecutar el siguiente comando para verificar el estado de los contenedores:
```Bash
docker ps -a
```

Acceso a pgAdmin: http://localhost:8081
